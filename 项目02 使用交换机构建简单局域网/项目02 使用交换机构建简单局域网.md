# 任务01 交换机的基本命令

### 查看用户视图

```
system-view
```

### 关闭信息中心

```
undo info-center enable
```

### 更改设备名称

```
sysname SW-1
```

### 查看当前配置

```
display current-configuration
```

也可以输入关键字母来执行，例如：

```
dis cur
```

### 查看 VLAN 信息

```
display vlan
```
